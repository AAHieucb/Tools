# Apache server 
1 phần mềm web server viết bằng C++, dùng mọi hđh. Nó hỗ trợ CGI, PHP, GZIP compression, virtual hosting, URL rewriting, xác thực và ủy quyền, caching, cân bằng tải.
Trong apache, mỗi request là 1 luồng. 
Apache server có httpd là 1 phần cài đặt cùng, nó là 1 http daemon xử lý request gửi tới.
Vd khi nhận request, apache server sẽ serve thư mục ở 1 port, hoặc execute 1 CGI script và trả lại kết quả.
